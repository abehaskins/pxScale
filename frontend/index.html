Using by Hand

pxScale can be used on any image URL by simply putting "pxscale.com/2x/" in front of the URL for 2x scaling, "pxscale.com/3x/" for 3x, etc.

For example, to scale "http://i.imgur.com/AdPCnGW.gif" simply navigate to "pxscale.com/3x/http://i.imgur.com/AdPCnGW.gif" to get the scaled version.
